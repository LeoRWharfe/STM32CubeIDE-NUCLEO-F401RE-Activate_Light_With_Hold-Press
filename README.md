# STM32CubeIDE-NUCLEO-F401RE-Activate_Light_With_Hold-Press
Simple script to activate the LD2 green light when hold-pressing user B1 (blue button). 
This script works for the NUCLEO-F401RE developement board.
Used addresses (Found in  official USer Manuel): https://www.st.com/resource/en/user_manual/um1724-stm32-nucleo64-boards-mb1136-stmicroelectronics.pdf
Compile using the official STM32CubeIDE software.
